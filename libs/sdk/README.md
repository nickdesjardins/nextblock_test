# @nextblock-cms/sdk

Typed block extensibility contract for NextBlock.

## Current Public Surface

The package exports the block authoring types defined in `src/lib/sdk.ts`,
including:

- `BlockContentSchema`
- `BlockData<TSchema>`
- `BlockProps<TSchema>`
- `BlockEditorProps<TSchema>`
- `BlockConfig<TSchema>`
